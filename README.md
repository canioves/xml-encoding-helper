# XML Encoding Helper

По умолчанию, в vscode открытие xml файлов происходит в кодировке UTF-8. Поэтому при открытии файлов с кодировкой Windows-1251, символы кириллицы отображаются некорректно. Для исправления, нужно вручную переоткрывать файл в нужной кодировке. Также, в vscode есть функция угадывания кодировки, но она неправильно работает с файлами в windows-1251.

Это расширение автоматически переоткрывает xml файлы с кодировкой windows-1251. Сейчас работает для xml и xsd файлов.
